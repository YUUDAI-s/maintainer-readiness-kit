# Code of Conduct

This project follows the Contributor Covenant spirit: be respectful, assume
good intent, and keep discussions focused on improving the software.

Unacceptable behavior includes harassment, personal attacks, publishing private
information, or repeatedly derailing maintainer work.

Project maintainers may edit, hide, or close comments and issues that violate
these expectations. Serious or repeated violations may result in a ban from the
project community.
